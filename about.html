OpenHanzi : pratiquer l'écriture des sinogrammes

OpenHanzi est l'application web interactive pour la pratique écrite du lexique présent dans le MOOC "Kit de contact en langues orientales : chinois".

Interface et utilisation

  • Informations lexicales : cliquer le sinogramme pour l'ordre correct et l'audio ; prononciation pinyin, définition, composition, étymologie ; liens de partage.
  • Pad d'écriture : zone blanche permettant de pratiquer l'écriture, une erreur sera suivit de la suggestion correcte.
  • Ecrit/Audio : cliquer pour jouer l'ordre d'écriture correct et l'audio.
  • Audio : cliquer pour jouer l'audio.
  • Effacer: cliquer pour effacer le pad d'écriture.
  • Sinogrammes colorés : votre niveau de maitrise d'un sinogramme est enregistré dans votre navigateur.
       - rouge : écriture non-maitrisée,
       - vert : écriture maitrisée.
